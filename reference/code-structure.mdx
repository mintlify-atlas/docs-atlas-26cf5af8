```java
            // Process conversion...

        }
    }
    sc.close();
}
```


  <Step title="Build API URL">

    ```java
    String url = "https://v6.exchangerate-api.com/v6/9ba310889b4d07769a662fc0/pair/"
    ```

  <Step title="Parse JSON">

    Parses JSON response into JsonObject

  <Step title="Extract Result">

## RegistroConversion (Data Record)

**Package:** `lad.com.alura.conversormoneda.modelos`

**Purpose:** Immutable data structure to store conversion history entries.

### Record Definition

```java
package lad.com.alura.conversormoneda.modelos;

import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

public record RegistroConversion(
    String monedaBase, 
    String monedaObjetivo, 
    double cantidad, 
    double resultado, 
    LocalDateTime fechaHora
) {
    @Override
    public String toString() {
        DateTimeFormatter formato = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm:ss");
        return "[" + fechaHora.format(formato) + "] " 
            + cantidad + " " + monedaBase + " =>> " 
            + resultado + " " + monedaObjetivo;
    }
}
```

### Record Components

<ParamField path="monedaBase" type="String">
  The base (source) currency code
</ParamField>

<ParamField path="monedaObjetivo" type="String">
  The target (destination) currency code
</ParamField>

<ParamField path="cantidad" type="double">
  The original amount in base currency
</ParamField>

<ParamField path="resultado" type="double">
  The converted amount in target currency
</ParamField>

<ParamField path="fechaHora" type="LocalDateTime">
  Timestamp when the conversion was performed
</ParamField>

### Java Records (Java 14+)

<Card title="What are Records?" icon="book">
  Records are a special kind of class introduced in Java 14 that provide:
  
  - Automatic constructor
  - Automatic getters (component methods)
  - Automatic `equals()` and `hashCode()`
  - Automatic `toString()` (can be overridden)
  - Immutability by default
</Card>

### Text Blocks (Java 13+)

The method uses **text blocks** for multi-line string literals:

<Tabs>
  <Tab title="Text Block Syntax">
    ```java
    System.out.print("""
        Multi-line
        String
        Content
        """);
    ```

    **Benefits:**
    
    No need for `\n` newline characters, no need for string concatenation, preserves formatting and indentation, and more readable for large text blocks.
  </Tab>
  <Tab title="Traditional Syntax (Pre-Java 13)">
    ```java
    System.out.print(
        "Sea bienvenido/a al Conversor de Moneda =]\n" +
        "\n" +
        "==================================================\n" +
        "1) Dólar           =>> Peso argentino\n" +
        "2) Peso argentino  =>> Dólar\n" +
        // ... more lines
    );
    ```

    **Drawbacks:**
    
    Requires `\n` for each line, requires `+` concatenation, less readable, and more error-prone.
  </Tab>
</Tabs>
